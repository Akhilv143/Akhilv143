# About Me

Currently undertaking my M.Sc. Bioinformatics internship. My work primarily focuses on analyzing bulk RNA-seq data using `limma` on combined count matrices for Differential Gene Expression (DGE), as well as Whole Exome Sequencing (WES) variant calling for both short and long-read data.<br><br>Open to collaborations involving translational cancer genomics, multi-omics data integration, and micro-RNA interactome research.<br><br>Seeking opportunities to expand my expertise in integrating Machine Learning and Deep Learning models into downstream clinical genetics workflows.<br><br>Continuously advancing my knowledge in long-read sequencing analysis techniques and the optimization of high-throughput computational pipelines.<br><br>Available for discussions regarding robust R scripting for transcriptomics, configuring WES pipelines, and managing bioinformatics environments within Linux (WSL).
